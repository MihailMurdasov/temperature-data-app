# Temperature Data App

GUI-приложение для хранения и анализа температурных измерений.

Проект реализует собственное красно-чёрное дерево для хранения значений температур и позволяет:
- добавлять температуры вручную;
- искать количество повторений заданной температуры;
- загружать данные из `.txt` файла;
- строить гистограммы по данным;
- разделять измерения по дням;
- строить гистограммы:
  - за конкретный день;
  - за все дни;
  - за последние 7 дней.


## 1. Обычный запуск

### Зависимости

Для Ubuntu/Debian:

```bash
sudo apt-get update
sudo apt-get install -y python3 python3-pip python3-tk tcl tk
python3 -m pip install -r requirements.txt
```

### Запуск приложения

Из корня репозитория:

```bash
cd src
python3 main.py
```

### Запуск тестов

```bash
cd src
pytest tests.py
```

### Вариант с venv

Если удобнее запускать проект в виртуальном окружении:

```bash
sudo apt-get update
sudo apt-get install -y python3 python3-venv python3-pip python3-tk tcl tk
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install matplotlib pytest
cd src
python3 main.py
```

## 2. Описание проекта

Программа предназначена для хранения температур окружающего воздуха и анализа статистики по измерениям.

Поддерживаемые возможности:
- добавление температур вручную;
- поиск числа повторений заданной температуры;
- загрузка температур из текстового файла;
- построение гистограммы по данным;
- разделение измерений по дням;
- построение гистограммы:
  - за конкретный день;
  - за все дни;
  - за последние 7 дней.
  

## 3. Как устроен проект

### Красно-чёрное дерево

В проекте вручную реализованы:
- класс `Node`;
- класс `RedBlackTree`;
- операции вставки;
- балансировка после вставки;
- повороты дерева;
- inorder-обход;
- поиск количества повторений значения.

Каждый узел хранит:
- значение температуры;
- цвет узла;
- ссылки на потомков и родителя;
- `count` — число повторений значения.

Благодаря этому одинаковые температуры не дублируются отдельными узлами, а накапливаются в счётчике.

### Графический интерфейс

GUI реализован через `tkinter`. Через интерфейс можно:
- вводить температуру вручную;
- искать количество повторений;
- переходить к следующему дню измерений;
- загружать данные из файла;
- строить гистограмму через `matplotlib`.

### Гистограмма

Гистограмма строится по пяти интервалам температур. Перед построением пользователь выбирает режим:
1. конкретный день;
2. все дни;
3. последние 7 дней.

После этого программа собирает отсортированные данные из дерева, считает частоты по диапазонам и строит график.


## 4. Формат входных данных

Для загрузки из файла ожидается обычный текстовый файл, где каждая строка содержит одно целое значение температуры.

Пример:

```text
18
19
19
21
17
22
```


## 5. Особенности и ограничения

- приложение рассчитано на запуск в графической среде;
- загрузка из файла ожидает только целые числа;
- файл с ошибочными строками вызывает сообщение об ошибке;
- гистограмма строится на основании 5 диапазонов, вычисляемых по текущим данным.
